# Security Policy

We take the security of this project seriously. Thank you for helping us keep the AI-Powered RAG Assistant secure for everyone.

---

## Supported Versions

Only the latest release of the RAG Assistant is actively supported with security updates. We encourage all users to stay updated to the latest version of the repository.

| Version | Supported |
| :--- | :--- |
| **Latest (main branch)** |  Yes |
| **All Older Versions** | ❌ No (Please upgrade) |

---

## Reporting a Vulnerability

**Please do not report security vulnerabilities via public GitHub issues.** Instead, help us keep your data safe by reporting them privately.

To report a vulnerability:
1. Email your findings to **sachindilshan040@gmail.com**.
2. Include a detailed description of the vulnerability, the potential impact, and step-by-step instructions or a proof-of-concept (PoC) to reproduce the issue.
3. We will acknowledge your report within **48 hours** and work on a resolution promptly.

We appreciate your cooperation in practicing **responsible disclosure** to give us time to patch the issue before making it public.
